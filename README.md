**jobbot3000** is a self-hosted, open-source job search copilot.

## Quickstart

Requires Node.js 20+.

```bash
npm install
npm run dev
# Open http://127.0.0.1:3100
```

For detailed setup, CLI usage, and environment options, see [docs/getting-started.md](docs/getting-started.md).
